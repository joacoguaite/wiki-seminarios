# Wiki de seminarios — Poesía + Narrativa

Wiki de conocimiento y espacio de preparación de clases para dos seminarios online de
**4 encuentros cada uno**: uno de **poesía** y otro de **narrativa**. Reúne las fuentes,
sus resúmenes, un banco de ejemplos citables, los planes y guiones de cada sesión, y las
diapositivas.

## Cómo está organizado

```
raw/        Fuentes originales (poemas, ensayos, crítica). Inmutable.
wiki/       Conocimiento procesado: fuentes, entidades, conceptos, ejemplos, seminarios, slides.
CLAUDE.md   Schema de trabajo: cómo se ingiere, arma clase, hacen slides y lints.
slide-style.md  Preferencias de diapositivas.
```

El detalle completo del flujo de trabajo (ingest, armado de clases, diapositivas Marp,
lint/síntesis y convenciones de páginas) vive en **[`CLAUDE.md`](./CLAUDE.md)**.
